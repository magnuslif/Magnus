Website
Pizzeria Santos

This is an advanced website for a madeup company. The website is written in Html5. The website is stylized and structured with Css3, Jquery and Bootstrap. The Css was simplified with Less. The website is userfriendly with an easy to navigate layout. The website is responsive and adapts to all resolution, all links and buttons are animated in some way to show that they are clickable and the images in the background has a real parallax to draw in the focus of the user. The website have also been made to work without JavaScript but of course looks better with it activated. The website is hosted via gitlab and has tests that run automatically whenever code is commited, thanks to CI/CD.

System
Lunch security system

This system was created to prevent people who are not from my school from eating in our lunch room. It is built with a Raspberry pi and is used to alarm the staff if someone is trying to get in to our lunchroom with an invalid chip. The program that reads the chips code and determines if it is valid or not is written in Python3. Bansed on if the chip is valid or not, the probram will display a red or green screen telling the user if they are allowed to eat or not, with a message telling them why.
